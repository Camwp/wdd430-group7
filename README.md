# Handcrafted Haven

**Handcrafted Haven** is a full-stack web application designed to connect artisans and crafters with customers who value high-quality handmade goods.  
The platform allows creators to showcase their work, manage their shop profiles, and sell their products in a beautiful and community-driven online marketplace.

### Purpose
To provide an accessible, sustainable, and user-friendly marketplace for handcrafted products—fostering a sense of community and supporting small artisans around the world.

### Tech Stack
- **Frontend:** Next.js (React)  
- **Backend:** Node.js + Express  
- **Database:** (to be finalized – SQLite / MongoDB / PostgreSQL)  
- **Styling:** Tailwind CSS or custom CSS  
- **Deployment:** Vercel  
- **Version Control:** Git & GitHub Projects (Kanban)  

### Design Goals
- Clean, responsive UI with cohesive branding  
- Accessible to all users (WCAG 2.1 AA standards)  
- Optimized for SEO and performance  
- Consistent typography, color palette, and layout grid

# Project Functional Requirements

### Seller Profiles:
- Authenticated artisans can create and manage personal profiles to display their work, share bios, and list products.

### Product Listings:
- Sellers can add items with images, descriptions, categories, and prices.
Users can browse, search, and filter listings by various criteria (e.g., category, price, rating).

### Reviews & Ratings:
- Authenticated users can post written reviews and rate products.

### User Authentication:
- Basic sign-up / log-in system to manage seller and buyer accounts.

### Responsive Design:
- Must adapt seamlessly across desktop, tablet, and mobile devices.

### Accessibility:
- Must meet WCAG 2.1 AA standards for accessible content.

### SEO & Performance:
- Follow modern best practices for site optimization and validation.

### Cloud Deployment:
- Host the app on Vercel with a shared GitHub repository and project board.
